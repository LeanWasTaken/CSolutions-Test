# CSolutions-Test
## Tools/Libraries used:
* Vue3 + Vuetify 3 - Frontend development
* Express - REST API
* Prisma (CLI, Studio) - Database creation and management



## How to start the project:

```sh
npm install
```

### Frontend

```sh
npm run dev
```

### Hot-Reload REST API

```sh
npm run server
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
